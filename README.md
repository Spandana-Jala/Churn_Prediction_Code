End-to-end customer churn prediction system using machine learning

Approach :

1.Data cleaning & feature engineering 2.RandomForest & XGBoost models 3.SHAP explainability 4.FastAPI inference

High churn among month-to-month contracts
Business Impact: 1.Enables proactive customer retention 2.Improves decision-making through explainable AI
